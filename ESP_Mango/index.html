Build pretty sites with websocket full duplex comunication

Creating projects for ESP32 and ESP8266 based boards has never been so easy. Desing your sites with images, JS scripts, CSS and all what you need. Declare all the functions and their parameters in ESP-Mango interface, and that's it! ESP-Mango takes care of all the code that you need to include in your Arduino sketch.
